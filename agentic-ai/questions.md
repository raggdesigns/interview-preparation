# Agentic AI questions

Notes and interview answers covering agentic AI foundations: LLMs, prompt engineering, tool-calling, structured outputs, Claude Code, MCP servers, sub-agents, RAG, and related production patterns.

The folder is split into two tracks:

- **`track_learning/`** — a curriculum to build real, hands-on understanding. Read it in order.
- **`track_interview/`** — how you actually phrase these topics in the interview room. Short, spoken-voice, senior framing.

Use the learning track to build depth. Use the interview track to rehearse delivery.
